# 📜 SKILL: Official Google MCP Servers

**Source:** `google/mcp` (GitHub)
**Purpose:** Authentic, Google-maintained connectors for AI Agents.

## 🚀 Remote Servers (Managed)

Accessible via endpoint, no local setup required.

- **BigQuery:** Data warehouse querying.
- **Compute Engine (GCE):** VM management.
- **Google Maps (Grounding Lite):** Location intelligence.
- **Google Security Operations (Chronicle):** Threat detection.
- **Kubernetes Engine (GKE):** Container orchestration.
- **Cloud Resource Manager:** Project metadata.
- **Developer Knowledge API:** Access to Google's technical docs.

## 📦 Open Source Servers (Self-Hosted)

Deploy locally or to Cloud Run.

- **Google Workspace:** Gmail, Calendar, Drive, Docs, Sheets, Slides.
- **Firebase:** App development platform.
- **Google Analytics:** Traffic analysis.
- **Google Cloud Storage (GCS):** Object storage.
- **Cloud Run:** Serverless deployment.
- **Genmedia:** Access to **Imagen** (Image Gen) and **Veo** (Video Gen).
- **Chrome DevTools:** Browser automation & debugging.
- **gcloud CLI:** Command-line interface wrapper.
- **Flutter/Dart:** Development tools.

## 🔒 Security Focused (`google/mcp-security`)

- **Security Command Center (SCC)**
- **Google Threat Intelligence (GTI)**
- **Chronicle SOAR**
