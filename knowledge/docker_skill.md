## 🧠 Core Capabilities

1. **Containerization:** Wrapping apps (Supabase, Edge Functions) in isolated boxes.
2. **Orchestration:** Using `docker-compose` to run AI + DB + App simultaneously.
3. **Dev Environments:** Creating reproducible dev environments for any machine.

## 🤝 Nangnoy's Usage Protocol

* **Supabase:** Run local Supabase instance via Docker for offline development.
* **Clean Up:** Run `docker system prune` weekly to save space.
* **Security:** Never bake secrets into Images. Use Environment Variables.
