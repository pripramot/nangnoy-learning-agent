# Presentation Master Skill (PowerPoint)

This skill empowers "Nangnoy" to generate professional, data-driven presentations programmatically.

## 1. Core Capabilities

### 🐍 Python Automation (`python-pptx`)

- **Objective:** Generate reports, summaries, and complex decks from raw data without opening PowerPoint.
- **Key Functions:**
  - `create_presentation(template_path)`: Initialize a deck from a corporate template.
  - `add_slide(layout_name)`: Insert slides based on master layouts.
  - `populate_table(data)`: valid JSON/CSV to PowerPoint tables.
  - `insert_chart(data, chart_type)`: Native charts from Pandas DataFrames.

### 📜 VBA Automation (Macro)

- **Objective:** Automate interactive tasks within an open presentation.
- **Key Scripts:**
  - `CleanUpFormatting`: Standardization of fonts and colors.
  - `ExportToPDF`: Automated export workflows.

## 2. Technical Implementation

### A. Python Example (Generate Annual Report)

```python
from pptx import Presentation
from pptx.util import Inches

def generate_report(data):
    prs = Presentation()
    slide = prs.slides.add_slide(prs.slide_layouts[0])
    title = slide.shapes.title
    title.text = "Annual Forensic Report"
    subtitle = slide.placeholders[1]
    subtitle.text = "Generated by Nangnoy AI"
    prs.save('report.pptx')
```

### B. "Canva-Style" Aesthetics

- Use **High-Quality Backgrounds** rather than solid colors.
- Use **Grid Layouts** for images (2x2, 3x3).
- **Typography:** Hierarchy is key (Title 44pt, Header 32pt, Body 18pt).
- **Icons:** Use SVG/PNG icons for visual cues (Shield, Map, Brain).

## 3. Best Practices

- **Templates:** Always prefer `.potx` templates over starting from scratch.
- **Data Binding:** Keep data separate from design (MVC pattern).
- **Accessibility:** Ensure Alt Text is generated for all added images.
