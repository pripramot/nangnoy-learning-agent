# Creator Agent Skill

This skill empowers AI agents to become multi-modal content creators, capable of generating high-quality articles, visual assets, audio scripts, and podcast outlines.

## 🎨 Creative Domains

### 1. Article & Text Creation

- **Structural Integrity**: Writing with clear headings, introductions, and conclusions.
- **Tone Modulation**: Adapting voice for technical, professional, or casual audiences.
- **Multilingual Synthesis**: High-quality translation and content localization (e.g., Thai/English).

### 2. Visual Asset Generation

- **Prompt Engineering**: Crafting precise descriptions for DALL-E, Midjourney, or Flux.
- **Image Enhancement**: Logic for improving resolution, style consistency, and branding.
- **Technical Diagrams**: Outlining schemas for Mermaid.js or SVG generation.

### 3. Audio & Podcasts

- **Scriptwriting**: Creating natural-sounding dialogue scripts for AI voice synthesis.
- **Show Notes**: Generating summaries, timestamps, and key takeaways for audio content.
- **Structure**: Defining intro, transitions, core segments, and outro sequences.

## 🛠 Integration Patterns

### Content Syndication

- **GitBook Integration**: Automated publishing of articles via API.
- **Markdown Mastery**: Using rich formatting to ensure content is "reader-ready".

### Media Pipeline

- **ElevenLabs/OpenAI TTS**: Patterns for voice generation.
- **Image APIs**: Direct integration with generative image providers.

## 🚀 Execution Flow: "Agent Create"

1. **Research**: Gather domain-specific facts (e.g., from VPC documentation).
2. **Synthesis**: Transform facts into a compelling narrative.
3. **Multi-modal Pass**: Generate accompanying badges (Shields.io), images, or scripts.
4. **Publish**: Deploy to destination (GitBook, GitHub, etc.).
