## 🔍 [01:00 - 04:00] เนื้อหาหลัก (Demo Use Case)

[cite_start]**บุญทิ้ง:** "สมมติว่ามีเคสเร่งด่วนที่ต้องการตรวจสอบประวัติการสื่อสารจากอุปกรณ์ที่เกี่ยวข้องกับคดีในพื้นที่อุดรธานีครับ ขั้นตอนแรกเราจะเริ่มที่การดึงข้อมูลจากซิมการ์ดก่อนครับ" [cite: 13, 14]

[cite_start]**นางน้อย:** "ใช่ค่ะ ในขั้นตอนนี้เราจะใช้ **SIM Card Extraction** โดยใส่ซิมลงในเครื่องอ่าน  [cite_start]ซึ่งระบบ Unicorn ของเราสามารถดึงข้อมูลได้ทั้งรายชื่อผู้ติดต่อ, ข้อความ SMS, และบันทึกการโทร เพื่อหาความเชื่อมโยงเบื้องต้นค่ะ" [cite: 15]

[cite_start]**บุญทิ้ง:** "จากนั้นเราจะขยับไปที่ **Device Extraction** ครับ ซึ่งรองรับทั้งระดับ Advanced และ Standard  [cite_start]โดยเราสามารถเลือกดึงเฉพาะไฟล์ที่ต้องการได้ เช่น รูปภาพวิดีโอ หรือระบบไฟล์ทั้งหมด (File System) เพื่อนำมาประกอบเป็นหลักฐานครับ" 

[cite_start]**นางน้อย:** "ความเจ๋งคือ ข้อมูลทั้งหมดที่ดึงมาจะถูกส่งเข้า **Shared Database (Postgres Docker)** ทันที  [cite_start]เพื่อให้หนูและร่างกู้คืนไฟล์ทำงานประสานกันได้ และเมื่อเสร็จสิ้น เราสามารถสร้างรายงานสรุปในรูปแบบ **PDF หรือ HTML** เพื่อส่งต่อให้ทีมสืบสวนได้เลยค่ะ"
